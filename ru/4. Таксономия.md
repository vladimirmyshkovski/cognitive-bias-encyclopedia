## Каталог по категориям

### 1. Суждение (judgment)

Оценка информации, аргументов, вариантов. Самая большая категория — здесь базовые «эвристики» Канемана-Тверски.

| Маркер | Искажение | Краткое описание |
|--------|-----------|------------------|
| ✅ | [[ru/5. Каталог/Anchoring]] | Цепляемся за первую информацию (якорь) |
| ✅ | [[ru/5. Каталог/Availability Heuristic]] | Судим по лёгкости примера |
| ✅ | [[ru/5. Каталог/Confirmation Bias]] | Ищем информацию, подтверждающую убеждения |
| ✅ | [[ru/5. Каталог/Framing Effect]] | Решение зависит от формулировки |
| ✅ | [[ru/5. Каталог/Loss Aversion]] | Потери болезненнее, чем выигрыши приятны |
| ✅ | [[ru/5. Каталог/Sunk Cost Fallacy]] | Продолжаем из-за вложенного |
| ✅ | [[ru/5. Каталог/Backfire Effect]] | Опровергающая информация укрепляет убеждение |
| ✅ | [[ru/5. Каталог/Status Quo Bias]] | Предпочитаем статус-кво |
| ✅ | [[ru/5. Каталог/Halo Effect]] | Одна черта = всё остальное |
| ✅ | [[ru/5. Каталог/Motivated Reasoning]] | Строим аргументы под желаемый вывод |
| ✅ | [[ru/5. Каталог/Curse of Knowledge]] | Эксперт не видит сложности для новичка |
| ✅ | [[ru/5. Каталог/Decoy Effect|Decoy Effect]] | Третий вариант меняет выбор между двумя |
| 🔴 | **Default Effect** | Принимаем предложенное по умолчанию |
| ✅ | [[ru/5. Каталог/Endowment Effect|Endowment Effect]] | Переоцениваем то, чем владеем |
| 🔴 | **Effort Justification** | Ценим результат по вложенному усилию |
| 🔴 | **Reactance** | Делаем наоборот при ограничении свободы |
| 🔴 | **Choice Overload** | Слишком много вариантов парализует решение |
| 🟡 | Belief Bias | Силу аргумента оцениваем по правдоподобию вывода |
| 🟡 | Conjunction Fallacy | Связка «A и B» кажется вероятнее, чем A |
| 🟡 | Contrast Effect | Суждение зависит от сравнения с другим |
| 🟡 | Mere Exposure Effect | Нравится то, что знакомо |
| 🟡 | Outcome Bias | Судим по исходу, не по процессу решения |
| 🟡 | Placebo Effect | Вера влияет на физический результат |
| 🟡 | IKEA Effect | Ценим то, что частично создали сами |
| 🟡 | Focusing Effect | Переоцениваем один аспект ситуации |
| 🟡 | Peak-End Rule | Помним пик и конец, а не длительность |
| 🟢 | Appeal to Novelty | «Новое = лучшее» |
| 🟢 | Appeal to Tradition | «Старое = проверенное» |
| 🟢 | Distinction Bias | Переоцениваем разницу между почти одинаковым |
| 🟢 | Functional Fixedness | Не видим новое использование знакомого |
| 🟢 | Weber-Fechner Bias | Субъективная разница не соответствует объективной |
| ⚪ | Authority Bias | [[Authority Bias\|см. социальное]] |
| ⚪ | Less-is-Better Effect | Частный случай Choice Overload |
| ⚪ | Decoy Effect (расшир.) | Перекрывается с Framing |

### 2. Память (memory)

Искажения воспоминаний — о прошлом помним неточно.

| Маркер | Искажение | Краткое описание |
|--------|-----------|------------------|
| ✅ | [[ru/5. Каталог/Hindsight Bias]] | «Я так и знал» — после события |
| ✅ | [[ru/5. Каталог/Misinformation Effect]] | Пост-событийная информация искажает память |
| 🔴 | **False Memory** | «Помним» события, которых не было |
| 🔴 | **Cryptomnesia** | Чужие идеи принимаем за свои |
| 🔴 | **Mandela Effect** | Коллективная «ложная память» о деталях |
| 🔴 | **Source Confusion** | Путаем источник информации |
| 🟡 | Rosy Retrospection | Прошлое кажется лучше, чем было |
| 🟡 | Reminiscence Bump | Лучше помним юность и молодость |
| 🟡 | Mood-Congruent Memory | Помним то, что совпадает с настроением |
| 🟡 | Serial Position Effect | Лучше помним начало и конец списка |
| 🟡 | Zeigarnik Effect | Лучше помним незавершённые задачи |
| 🟡 | Spacing Effect | Распределённое обучение запоминается лучше |
| 🟢 | Telescoping Effect | Сдвигаем события во времени (ближе к «сейчас») |

| 🟢 | Level-of-Processing Effect | Глубокая обработка = лучшее запоминание |
| 🟢 | State-Dependent Memory | Лучше помним в том же состоянии |

## Принципы отбора

**Добавляем запись**, если:
- Искажение **значимо** (есть публикации, есть в Wikipedia)
- **Не перекрывается** с существующим (если перекрывается — обновляем существующее)
- **Релевантно** для нашей аудитории (русскоязычные читатели, интересующиеся саморазвитием и мышлением)
- Есть **минимум 2 академических источника**

**Не добавляем**, если:
- Полностью перекрывается с существующим (см. ⚪ маркеры)
- Экзотическое / редко цитируемое
- Спорное в академической литературе
